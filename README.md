# 🧬 Inner_ANIMALSOUL_Dectector

> A fully offline, single-page personality analysis tool powered by a 48-animal psychological database.

## What is this?

**Archetype** is a web app that maps your personality through the lens of animal archetypes. Select 2–4 animals that resonate with you and the engine synthesizes a deeply personalized psychological profile across **8 analytical pillars**.

No accounts. No backend. No data sent anywhere. 100% offline.

---

## Features

- **48 Animals** across 4 biological categories (Land Mammals, Birds, Reptiles & Insects, Aquatic & Amphibians)
- **Single Analysis Mode** — Select 2–4 animals to generate your synthesized persona
- **Dual Comparison Mode** — Build two separate profiles and compare them side-by-side
- **8-Pillar Deep Analysis:**
  1. 🧠 Core Mindset
  2. 🛡️ Emotional Architecture
  3. 💼 Operational Style
  4. 🎭 The Shadow Side
  5. ⚡ Cognitive Cadence
  6. 🤝 Relational Architecture
  7. 🔥 Existential Triggers
  8. 🌐 Network Positioning
- **Volatile Flux Detection** — Identifies when opposing animal instincts create internal paradoxes
- **Comparison Reports** — Full 8-pillar differential analysis showing where two profiles converge and diverge
- **Retro Arcade UI** — Dark mode, 8-bit Press Start 2P font, flat pixel-border aesthetic
- **Fully Mobile Responsive** — Works seamlessly on phones and tablets

## How to Use

1. Download or clone this repository
2. Open `index.html` in any modern browser
3. No installation, no dependencies, no internet required

```bash
git clone https://github.com/YOUR_USERNAME/archetype-taxonomy-engine.git
cd archetype-taxonomy-engine
# Open index.html in your browser
```
